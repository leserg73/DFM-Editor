Ctrl+N: Новая форма
Ctrl+O: Открыть DFM
Ctrl+Shift+R: Перезагрузить текущий DFM файл
Ctrl+Shift+C: Пакетное преобразование (bin<>txt-dfm)
Ctrl+R: Загрузить ресурс

Ctrl+B: Переключить в двоичную форму
Ctrl+S: Сохранить DFM-файл
Ctrl+Alt+S: Сохранить вывод
Ctrl+Shift+S: Сохранить в двоичный файл
Ctrl+U: Создать PAS-файл (unit)
Ctrl+G: Создать код выполнения

Ctrl+P: Предварительный просмотр формы
Ctrl+Shift+P: Предварительный просмотр формы используя вывод
Ctrl+H: Переключатель "Показывать скрытые"
Ctrl+E: Переключатель "Включить отключенные элементы управления"

Ins: Добавить объект
Ctrl+Ins: Добавить свойство
Shift+Ins: Добавить запись
Enter: Правка
Delete: Удалить

Ctrl+Up: Переместить выше
Ctrl+Down: Переместить ниже
Ctrl+Shift+Up: Переместить наверх
Ctrl+Shift+Down: Переместить вниз

Ctrl+Shift+U: Удалить лишние свойства

Ctrl+I: Проверить обновление

F1: Показать справку
Shift+F1: Показать диалог "О программе"

Ctrl+Alt+O: Настройки

F3: Продолжить поиск
Ctrl+A: Выделить весь текст

В дизайнере формы:
Ctrl+C: Копировать выбранный элемент управления
Ctrl+X: Вырезать выбранный элемент управления
Ctrl+V: Вставить скопированный/вырезанный элемент управления

В инспекторе свойств:
Enter: Установить значение
Esc: Сбросить значение
Tab: Перейти к следующему свойству
Shift+Tab: Перейти к предыдущему свойству
